B.Tech. Final Year Project
August 2015 - May 2016
Capturing behavioral traits based on mobile usage characteristics for efficient recommendation engine

The main objective of the project was to show that the mobile usage behavior can capture various behavioral traits of a person, namely Social Behavior, Financial Behavior, Psychological Behavior and hence can be used for targeted advertisements
Developed an android application for collecting user's call log and user queries and website/app visits. The data obtained from the user and along with the ground truth data is processed, and then used in the classifier for each behavioral class.
System design was divided into five parts: Data Logging, Storage, Ground Data, User Data Processing, Classification. The technologies used were Android for data logging and ground data collection, Amazon S3 and Amazon EC2 for storage and server and Python Rake and NLTK for processing and classication.
Initial results found showed that user's behaviour was reflected from different mobile usage characterstics. Furthur work would be carried on by another student of the college.
